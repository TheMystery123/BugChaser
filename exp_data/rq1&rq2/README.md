# RQ1 & RQ2 数据

## 概览

本研究的数据集包含71个应用程序，总共有121个崩溃错误。这些应用程序来自两个主要的数据集：来自Themis基准测试的20个应用程序，其中包含34个错误，以及从F-Droid收集的额外51个应用程序，其中包含87个错误。

## 数据集详情

### 1. Themis基准测试数据集

你可以在[Themis基准测试GitHub仓库](https://github.com/the-themis-benchmarks/home)下载数据集。

### 2. 额外收集的数据集
